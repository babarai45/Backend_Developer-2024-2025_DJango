Django Commands: 
1. To create a Django project: django-admin startproject project_name
2. To create a Django app: python manage.py startapp app_name
3. To run the server: python manage.py runserver
4. To create a superuser: python manage.py createsuperuser
5. To migrate the database: python manage.py migrate
6. To make migrations: python manage.py makemigrations
7. To open the shell: python manage.py shell
8. To open the database shell: python manage.py dbshell
9. To collect static files: python manage.py collectstatic
10. To run the test cases: python manage.py test
